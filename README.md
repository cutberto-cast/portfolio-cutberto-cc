# Portfolio Profesional - Cutberto Castillo

![Portfolio Preview](/public/og-image.png)

Portfolio moderno desarrollado con Next.js 14, TypeScript, Tailwind CSS y Three.js para experiencias 3D interactivas.

## Características

- **Next.js 14** con App Router
- **Tailwind CSS** con animaciones personalizadas
- **Three.js** y **React Three Fiber** para gráficos 3D
- **Totalmente responsive**
- **Modo oscuro** automático
- **SEO optimizado**
- **Analytics** integrado
- **Performance 90+** en Lighthouse

## Tecnologías

- **Framework:** Next.js 14
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **3D:** Three.js, @react-three/fiber, @react-three/drei
- **UI:** Lucide React, Framer Motion
- **Deploy:** Vercel

## Cómo ejecutar localmente

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/portfolio.git
cd portfolio

# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm run dev
